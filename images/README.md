# Images

This directory contains optimized images for the InstintiaVet website.

## Guidelines

- All images must be royalty-free (Unsplash, Pexels, Creative Commons or similar)
- Format: WebP preferred, JPEG as fallback
- Maximum size: 200KB per image
- Minimum resolution: 400x300px
- Maximum total images: 10
- Document all sources in `/ATTRIBUTIONS.md`
